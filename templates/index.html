Camp Fire LandSat-8 Infrared Maps

I was inspires to create this map because of some images circulating around the internet. Landsat-8 a satellite shared by NASA and USGS. On November 8, 2018 at 10:45AM about 4 hour and 15 minutes after the start of the fire LandSat took this image.

From visibleearth.nasa.gov

From cnet.com

Most New Sources are circulating this image with the text removed, ignoring all context for this photo. Red added to the photo is from an infrared band camera aboard the satelitte. Without context it looks like these are flames visible from space. Below is an interactive map of the highest regions of infrared measured by the satelitte in this picture. I am not a professional this data should not be used for anything other than insight. Below are the steps I took to create this data to maintain transparency.

How it Was Made

The original satellite image was obtained from Earth Explorer a source for satillite images from the USGS and NASA. The polygon shape was created with QGIS the steps I took were to:

  • Use Georefrencer to align the image to GPS positions
  • Using the infrared band I created a mask of values greater than 105 (107 max)
  • Ran Polygonize to convert to shape file
  • Split the vector Layer based on Mask Value
  • Manually removed shapes not related to the fire
  • Exported to geojson
This was my first time using QGIS which is why I say to take this information with a grain of salt. If you have expierence in this field and can confirm or correct my workflow I would appreciate it.
